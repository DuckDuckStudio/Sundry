- ✕ 弃用的配置文件版本，新版本的 Sundry 不再支持。

## 关于配置文件的说明

### 配置文件放在哪里？
在 **1.1.0** 之前，配置文件在程序目录下的 `config.json` 中；  
在 **1.1.0** 之后，配置文件在用户目录下的 `.config/DuckStudio/Sundry/config.json` 中。

### 卸载时会删除配置文件吗？
~~由于作者实在是太懒了~~，在卸载 Sundry 时**不会**移除配置文件，如果需要可以手动移除。  

### 怎么从旧版配置文件迁移/更新至新版配置文件？
请尝试运行 `sundry config update` 命令。  
如果更新失败，请重新运行 `sundry config init` 初始化新的配置文件。
