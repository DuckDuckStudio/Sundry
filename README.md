# Sundry

Sundry 是一个自动扫描和移除无效 winget 软件包的工具。  
此工具的想法与处理逻辑来源于 [CoolPlayLin/AutoPublish](https://github.com/CoolPlayLin/AutoPublish) 项目。  
此工具使用 C# .NET 8 编写，依赖以下库：  
- [YamlDotNet - MIT](https://github.com/aaubry/YamlDotNet)

项目 Docker 容器为 Linux。
